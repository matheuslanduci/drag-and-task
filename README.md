## Como rodar o projeto

### Pré-requisitos

- [Bun 1.1.31](https://bun.sh/)

### Passo a passo

- Clone o repositório

```bash
$ git clone https://github.com/matheuslanduci/drag-and-task
```

- Instale os pacotes

```bash
$ bun install
```

- Inicie o projeto

```bash
$ bun run dev
```

- O projeto estará rodando em http://localhost:5173
